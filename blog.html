Here Are Some Questions with Answers

1. Difference between var, let and constant

Ans : Var is globally scoped or function scoped. It can be reassigned . let is globally scoped or block scoped and it can be reassigned on the other hand const is global scoped or block scoped and it cannot be reassigned.

2. Difference between Regular function and Arrow function

Ans :In Regular function we have to use return key word to return any value. If Arrow function contains one expression,you can omit the carly braces and then the expression will be implicity returned.Regular function have their own 'this'. But arrow function don't have their own 'this'.

3. Why we need to use template string in javascript

Ans :A template string, or called template literal, is a practical and useful syntax in JavaScript. It makes our programs neater and better when it comes to string handling.
